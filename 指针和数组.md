## 数组\(array\)和向量\(vector\)

### 数组:

数组是有类型属性的。同一数组中的每一个元素都必须属于同一数据类型。一个数组在内存中占一片连续的存储单元。

例子:

```
int array[100];
```

```
#include <iostream>

const int SIZE = 10;
int main()
{
    int array[SIZE];
    for(int i = 0; i<SIZE; ++i){
        array[i]=i; 
    }
    for(int i = 0; i<SIZE; ++i){
        std::cout<<array[i]<<std::endl;    
    }

```
